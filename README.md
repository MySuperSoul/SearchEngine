# SearchEngine

项目实训课程项目, Techhub知识库搜索引擎。通过收集各大博客网站、技术问答网站的技术文档、博客、视频等多元异构的数据，搭建后台的知识库，作为搜索引擎的数据支撑。

<br>

### 项目结构

- `front-end`: 前端子系统
- `back-end`: 后端子系统
- `crawler`: 爬虫子系统
- `docs`: 项目文档
